# Travel-App-AI

**AI Travel Planner is a full-stack web application that generates personalized travel itineraries using AI.
Users can enter a destination, trip duration, and budget, and instantly receive a detailed, day-wise travel plan with places to visit, food suggestions, and local tips.

## How it Works

1.User enters destination, number of days, and budget
2.Frontend sends a POST request to the backend
3.Backend constructs a prompt and sends it to the AI model
4.AI generates a detailed travel itinerary
5.The response is displayed in a styled, readable format on the UI


## Tech Stack

Frontend:
- React JS
- Bootstrap
- Fetch API (no Axios)
- useState Hook


Backend:
- Node.js
- Express
- MongoDB
- Mongoose
- dotenv
- AI API(LLM- baseditineray generation)
